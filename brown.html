Using React, I created a list website that would aggregate some value- I decided to create a shopping interface for Brown Dining Services (BDS).

I was inspired by Brown's dining options. During quarantine, my friends and I had a spirited argument trying to rank them.



My website aims to provide a way for students to view and order their food, which may not require as much in-person interaction.



Design


The project required filter and sorting functions. I implemented a Location and Meal filter. I also enabled sorting by price.



Code Overview




Usability Principles


  • Logical page setup
  • Page is responsive
  • Indicators for what settings are selected
  • Text indicates if no items fit parameters or if cart is empty
  • Buttons respond to hover
  • Colors chosen for adequate contrast
  • Distinct item cards
  • Padding and margins to keep content within high visibility region


Item Illustrations

I used Procreate to create illustrations for each item available for purchase.

Conclusion


After getting used to React, making this website was really enjoyable. The HTML for this project only contained one element, so I think experimenting more with using HTML and React would be helpful in the future.

I also had so much fun designing the website. I enjoyed "ordering food", and am excited to learn more of what's possible with React.
